Today's Tip:
A function has been added to the manner which changes a subclass through a NPC, so that one can change a class through the Character Information window (Alt + T). You can freely change classes in all regions except some certain regions. However, when changing classes, it becomes a status where the combat ability decreases and skills and items cannot be used for a certain time due to the 'Identity Crisis' debuff effect.
